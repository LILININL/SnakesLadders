# Monopoly Frontend

โฟลเดอร์นี้เตรียมไว้สำหรับ asset/script เฉพาะเกมเศรษฐี

โครงโฟลเดอร์:
- `assets/` ภาพ/เสียง/ไอคอนของเกมเศรษฐี
- `styles/` CSS เฉพาะเกมเศรษฐี
- `js/` logic client เฉพาะเกมเศรษฐี

สถานะปัจจุบัน:
- เกมเศรษฐีเปิดใช้งานแล้วใน game list
- หน้าห้องยังใช้ shell UI กลาง (`wwwroot/games/snakes-ladders/js`) และแสดงตารางเศรษฐีผ่าน renderer กลาง
