# Monopoly Module

โมดูลเกมเศรษฐีที่ใช้งานได้แล้วผ่าน `IGameRoomModule`

ไฟล์หลัก:
- `Domain/MonopolyDefinitions.cs` ค่าคงที่เกม (เงินตั้งต้น, ช่องคุก, ผ่าน GO)
- `Domain/MonopolyModels.cs` โครงข้อมูลห้องเกมเศรษฐี + template ตาราง 40 ช่อง
- `Services/MonopolyGameRoomModule.cs` logic สร้างห้อง/เริ่มเกม/เดินเทิร์น/จบเกม

mechanics พื้นฐานที่มีแล้ว:
- ซื้อทรัพย์สิน/ซื้อต่อผ่าน action prompt
- จ่ายค่าเช่าให้เจ้าของ
- จ่ายภาษี/ค่าปรับตามเศรษฐกิจเกม
- Chance / Community Chest แบบสุ่มเหตุการณ์พื้นฐาน
- Go To Jail / ข้ามตาในคุก
- ล้มละลาย + คืนทรัพย์สินเข้าธนาคารหรือโอนตอนแพ้จริง
- หนี้ผู้เล่นแก้ด้วยจำนอง/รื้อสิ่งปลูกสร้างก่อน ไม่ขายเมืองโอนเต็มราคาเพื่อรอด
- Sudden-death rent scaling หลังกลางเกม
- Final Duel 6 รอบเมื่อห้องที่เริ่ม 3+ คนเหลือ 2 คนสุดท้าย
- จบเกมแบบ Last Standing หรือหมดรอบแล้วตัดสินจากมูลค่าสุทธิ

## ค่าหลักของเกมตอนนี้

- เงินเริ่มต้น: `฿1,200`
- ผ่าน `GO`: รับ `฿150`
- ขายคืนธนาคาร: base `30%` ของราคาเมืองปัจจุบัน บวกเงินคืนจากสิ่งปลูกสร้างตามระดับ
- ค่าปรับออกคุกเริ่มต้น: `฿150`
- หลังรอบ `10`: ค่าเช่าเข้าสู่ช่วง `Sudden Death` และโตแรงขึ้นอีก
- Final Duel:
  - Trigger: เฉพาะห้องที่เริ่ม `3+ คน` แล้วเหลือ `2 คน`
  - Duration: `6 รอบ`
  - GO: รอบ 1-2 รับ `฿60`, รอบ 3-6 รับ `฿0`
  - Rent bonus: `+40% / +60% / +85% / +110% / +140% / +200%`
  - ราคาเมือง: หยุดโตทันที
  - ขายคืนธนาคารช่วง Duel: base เหลือ `15%`
  - ถ้าครบ Duel ทั้ง 6 รอบ: ตัดสินด้วย `Net Worth > Cash > seat`

## กติกาหนี้ / แพ้

- แพ้จริง = `ล้มละลาย`
- ถ้าเป็นหนี้ธนาคาร:
  - ใช้ `รื้อสิ่งปลูกสร้าง`
  - ใช้ `จำนอง`
  - ใช้ `ขายคืนธนาคาร`
  - ถ้ายังไม่พอ ค่อยล้มละลาย
- ถ้าเป็นหนี้จากค่าผ่านทางผู้เล่น:
  - ใช้ `รื้อสิ่งปลูกสร้าง`
  - ใช้ `จำนอง`
  - ไม่เปิดขายเมืองโอนเต็มราคาให้เจ้าหนี้เพื่อรอด
  - ถ้ายังปิดหนี้ไม่ได้ ระบบจะตัด `ล้มละลาย`
- ถ้าเข้าช่วง `Final Duel`
  - ซื้อจากธนาคารยังได้
  - ซื้อต่อจากผู้เล่นยังได้
  - Landmark ยังห้ามซื้อต่อ
  - จำนอง/ไถ่ถอนยังใช้กติกาปกติในรอบแรกของระบบนี้

## ปุ่มใน Action UI

### 1) ปุ่มทอย

- `ทอยเต๋า`
  - เงื่อนไข: ถึง phase `AwaitRoll`
  - ผล: ทอย 2 ลูกและเดินตามแต้ม
  - หมายเหตุ:
    - ได้ `ดับเบิ้ล` = ได้เล่นต่อ
    - ได้ `ดับเบิ้ล 3 ครั้งติด` = เข้าคุกทันทีและไม่เดิน

### 2) ปุ่มตอนติดคุก

- `จ่ายค่าประกันออกจากคุก`
  - เงื่อนไข: อยู่ phase `AwaitJailDecision`
  - ผล: จ่ายค่าปรับแล้วออกจากคุก กลับไป phase ทอย
  - ถ้าเงินไม่พอ: ระบบจะพาไปช่วงจัดการทรัพย์เพื่อหาเงินก่อน

- `เสี่ยงทอยดับเบิลเพื่อออกจากคุก`
  - เงื่อนไข: อยู่ phase `AwaitJailDecision`
  - ผล:
    - ถ้าทอยได้ดับเบิ้ล = ออกจากคุกและเดินตามแต้ม
    - ถ้าไม่ได้ดับเบิ้ล = ยังอยู่คุก ไม่เดิน และค่าปรับรอบถัดไปคูณ 2

### 3) ปุ่มตอนซื้อทรัพย์

- `ซื้อทรัพย์สิน`
  - เงื่อนไข: ตกช่องธนาคาร เงินพอ
  - ผล: จ่ายเงินแล้วช่องเป็นของคุณ

- `ซื้อทรัพย์สินนี้ต่อ`
  - เงื่อนไข: ตกบ้านคนอื่น จ่ายค่าผ่านทางแล้ว ช่องนั้นยังไม่เป็นแลนด์มาร์ก และเงินพอ
  - ผล: จ่ายเงินให้เจ้าของเดิม แล้วเปลี่ยนเจ้าของทันที

- `ข้ามการซื้อ` / `ไม่ซื้อ / จบช่วงนี้`
  - เงื่อนไข: อยู่ช่วงตัดสินใจซื้อ
  - ผล: ไม่ซื้อ แล้วไปช่วงจัดการทรัพย์สินหรือจบช่วงนั้นต่อ

### 4) ปุ่มเมนูจัดการทรัพย์สิน

ปุ่มในหมวดนี้บางปุ่มเป็นแค่การเปิด step ถัดไป ยังไม่ส่ง action ไป server ทันที

- `อัปเกรดเมือง / โรงแรม / แลนด์มาร์ก`
  - เงื่อนไข: มีช่องที่อัปเกรดได้ในเทิร์นนี้
  - ผล: เปิดลิสต์ให้เลือกช่องที่จะอัปเกรด

- `ลดระดับเมือง / ขายโรงแรม / รื้อแลนด์มาร์ก`
  - เงื่อนไข: มีสิ่งปลูกสร้างที่ขายลดระดับได้
  - ผล: เปิดลิสต์ให้เลือกช่อง

- `จำนองทรัพย์สิน`
  - เงื่อนไข: มีทรัพย์ที่จำนองได้
  - ผล: เปิดลิสต์ให้เลือกช่อง

- `ไถ่ถอนจำนอง`
  - เงื่อนไข: มีทรัพย์ติดจำนองและเงินพอไถ่
  - ผล: เปิดลิสต์ให้เลือกช่อง

- `จบเทิร์น`
  - เงื่อนไข: อยู่ phase `AwaitManage` หรือ `AwaitEndTurn`
  - ผล: ส่งเทิร์นให้คนถัดไป
  - หมายเหตุ: ถ้ายังมีหนี้หรือเงินติดลบ ระบบอาจตัด `ล้มละลาย`

- `ประกาศล้มละลาย`
  - เงื่อนไข: ใช้ได้ตอนอยู่สถานะที่เกมเปิดให้จัดการหนี้/ทรัพย์
  - ผล: ยอมแพ้ทันที ออกจากการแข่งขัน และระบบจัดการทรัพย์ตามกติกา

- `ย้อนกลับ`
  - ผล: กลับไปเมนูจัดการทรัพย์สิน ไม่ส่ง action ไป server

### 5) ปุ่มลัดอัปเกรดทันที

- `อัปเกรดเมืองนี้ทันที`
- `สร้างโรงแรมที่นี่ทันที`
- `สร้างแลนด์มาร์กที่นี่ทันที`
  - เงื่อนไข: ระบบเปิดสิทธิ์ให้ช่องนั้นในเทิร์นนี้ และเงินพอ
  - ผล: อัปเกรดขึ้น 1 ขั้นทันที
  - ข้อจำกัด: 1 เทิร์นอัปเกรดได้แค่ `1 ครั้ง`

### 6) ปุ่มในลิสต์อัปเกรด

- `build-property`
  - เงื่อนไข:
    - ต้องเป็นของคุณ
    - ต้องอยู่ช่วงจัดการทรัพย์
    - ช่องนั้นต้องอยู่ใน `upgradeEligibleCellIds`
    - เทิร์นนี้ยังไม่ใช้อัปเกรดไปแล้ว
    - ช่องห้ามติดจำนอง
    - เงินพอ
    - stock บ้าน/โรงแรมยังเหลือ
  - ผล:
    - บ้านไม่ครบ 4 = สร้างบ้าน +1
    - ครบ 4 บ้าน = อัปเป็นโรงแรม
    - มีโรงแรมแล้ว = อัปเป็นแลนด์มาร์ก

### 7) ปุ่มในลิสต์ขายสิ่งปลูกสร้าง

- `sell-structure`
  - เงื่อนไข:
    - ต้องเป็นของคุณ
    - ต้องอยู่ช่วงจัดการทรัพย์
    - ต้องผ่านกฎ `Even Build`
  - ผล:
    - รื้อแลนด์มาร์ก
    - หรือขายโรงแรมกลับเป็น 4 บ้าน
    - หรือขายบ้านออก 1 หลัง
    - ได้เงินคืนบางส่วน

### 8) ปุ่มในลิสต์จำนอง

- `mortgage-property`
  - เงื่อนไข:
    - ต้องเป็นของคุณ
    - ยังไม่ติดจำนอง
    - ไม่มีบ้าน/โรงแรม/แลนด์มาร์กอยู่บนช่อง
  - ผล:
    - ได้เงินสดทันทีประมาณ 50% ของราคาเมืองปัจจุบัน
    - ช่องนั้นหยุดเก็บค่าผ่านทาง
    - ช่องนั้นอัปเกรดไม่ได้จนกว่าจะไถ่ถอน

### 9) ปุ่มในลิสต์ไถ่ถอนจำนอง

- `unmortgage-property`
  - เงื่อนไข:
    - ต้องเป็นของคุณ
    - ช่องนั้นติดจำนองอยู่
    - เงินพอ
  - ผล:
    - จ่ายคืนธนาคารประมาณ 110% ของมูลค่าจำนอง
    - ช่องกลับมาเก็บค่าผ่านทางได้ตามปกติ

### 10) ปุ่มในลิสต์ขายคืนธนาคาร

- `sell-property`
  - เงื่อนไข:
    - ใช้ตอนอยู่ช่วงแก้หนี้หรือเงินติดลบ
    - ถ้าเป็นหนี้ผู้เล่น ปุ่มนี้จะไม่เปิดให้ใช้หนีหนี้ตรง ๆ
  - ผล:
    - ขายช่องคืนธนาคาร
    - ช่องกลับเป็นแปลงว่าง
    - ผู้เล่นอื่นมาตกซื้อใหม่ได้
    - ถ้าอยู่ใน `Final Duel` ราคาขายคืนจะลดลงอีกเหลือ base `15%`

## ความหมายของคำสำคัญ

- `จำนอง`
  - เอาทรัพย์ไปแลกเงินสดชั่วคราวจากธนาคาร
  - ผลเสียคือช่องนั้นหยุดทำรายได้ และยังอัปเกรดไม่ได้

- `ไถ่ถอน`
  - จ่ายเงินคืนธนาคารเพื่อปลดจำนอง
  - หลังจากนั้นช่องกลับมาทำงานปกติ

- `ซื้อต่อ`
  - ซื้อสิทธิ์ความเป็นเจ้าของต่อจากเจ้าของเดิมหลังคุณจ่ายค่าผ่านทางแล้ว
  - ใช้ไม่ได้กับช่องที่เป็นแลนด์มาร์ก

- `ล้มละลาย`
  - ยอมแพ้หรือถูกระบบตัดแพ้เพราะปิดหนี้ไม่ไหว
  - ออกจากเกมทันที

- `Even Build`
  - เวลาขายลดระดับสิ่งปลูกสร้าง ต้องขายแบบสมดุลในชุดสี

## หมายเหตุสำคัญ

- อัปเกรดได้แค่ `1 ครั้งต่อเทิร์น`
- อัปเกรดได้เฉพาะช่องที่ระบบเปิดสิทธิ์ให้ในเทิร์นนั้น
- ช่องติดจำนอง `ห้ามอัปเกรด`
- ช่องมีสิ่งปลูกสร้าง `ห้ามจำนอง`
- ช่องเป็นแลนด์มาร์ก `ห้ามซื้อต่อ`
- ถ้ากด `จบเทิร์น` ทั้งที่ยังมีหนี้ ระบบอาจตัด `ล้มละลาย` ทันที
- ถ้าเกมเริ่มต้นมาแค่ `2 คน` จะไม่เข้า `Final Duel` และยังใช้กติกาปิดเกมปกติ
